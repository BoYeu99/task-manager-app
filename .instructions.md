# Hướng Dẫn Copilot - Ứng Dụng Quản Lý Công Việc

## Tổng Quan Dự Án
Đây là ứng dụng full-stack quản lý công việc được xây dựng với Node.js/Express backend và vanilla JavaScript frontend.

## Tech Stack
- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Cơ Sở Dữ Liệu**: Lưu trữ trong bộ nhớ (có thể mở rộng để sử dụng cơ sở dữ liệu thực)
- **API**: RESTful API với JSON endpoints

## Kiến Trúc

### Backend (server.js)
- Express server chạy trên cổng 3000
- RESTful API endpoints cho quản lý công việc
- Lưu trữ công việc trong bộ nhớ với UUID identifiers
- CORS được bật để giao tiếp với frontend

### Frontend (public/)
- Single-page application với vanilla JavaScript
- Lọc công việc theo thời gian thực (Tất Cả, Đang Thực Hiện, Hoàn Thành)
- UI responsive dựa trên gradient CSS
- Sử dụng Fetch API để giao tiếp với server

## API Endpoints

### GET /api/tasks
Trả về mảng JSON chứa tất cả công việc.

### POST /api/tasks
Tạo công việc mới. Yêu cầu JSON body:
```json
{ "title": "Mô tả công việc" }
```

### PATCH /api/tasks/:id/toggle
Chuyển đổi trạng thái hoàn thành của công việc.

### DELETE /api/tasks/:id
Xóa công việc theo ID.

## Tính Năng Chính
1. **Thêm Công Việc** - Tạo công việc mới với tiêu đề mô tả
2. **Đánh Dấu Hoàn Thành** - Đánh dấu công việc đã hoàn thành/đang thực hiện
3. **Xóa Công Việc** - Xóa vĩnh viễn công việc
4. **Lọc Công Việc** - Xem tất cả, đang thực hiện, hoặc hoàn thành
5. **Giao Diện Responsive** - Hoạt động trên desktop và mobile

## Hướng Dẫn Phát Triển

### Phong Cách Code
- Sử dụng tên biến và hàm có ý nghĩa
- Giữ hàm tập trung vào một trách nhiệm duy nhất
- Thêm comment cho logic phức tạp
- Sử dụng indentation nhất quán (2 spaces)

### Phát Triển Frontend
- Giữ DOM manipulation hiệu quả
- Escape user input để ngăn XSS attacks
- Sử dụng CSS classes cho styling thay vì inline styles
- Xử lý lỗi API một cách graceful

### Phát Triển Backend
- Validate tất cả user inputs
- Sử dụng HTTP status codes phù hợp (201 cho created, 404 cho not found, etc.)
- Trả về consistent JSON response formats
- Xử lý lỗi một cách graceful

## Bắt Đầu

1. Cài đặt dependencies: `npm install`
2. Chạy server: `npm start`
3. Mở http://localhost:3000 trong trình duyệt
4. Bắt đầu quản lý công việc!
