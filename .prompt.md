# Mẫu Prompt Copilot - Ứng Dụng Quản Lý Công Việc

Bạn đang làm việc trên ứng dụng quản lý công việc. Đây là dự án full-stack có:
- **Backend**: Node.js/Express REST API
- **Frontend**: HTML/CSS/JavaScript single-page application
- **Tính Năng**: Thêm, xóa, chuyển đổi hoàn thành, lọc công việc

## Ngữ Cảnh Chính

### Cấu Trúc Dự Án
```
task-manager-app/
├── server.js              # Express API server
├── package.json           # Node.js dependencies
├── public/
│   ├── index.html         # Frontend entry point
│   ├── styles.css         # Styling
│   ├── app.js             # Frontend logic
```

### Triển Khai Hiện Tại
- Lưu trữ công việc trong bộ nhớ (không có cơ sở dữ liệu)
- RESTful API với UUID task IDs
- Frontend giao tiếp qua Fetch API
- Lọc và render theo thời gian thực

## Khi Trợ Giúp Dự Án Này

1. **Duy Trì Nhất Quán**: Tuân theo các pattern và style code hiện có
2. **Khả Năng Tương Thích API**: Giữ các endpoints hiện có hoạt động trừ khi được yêu cầu thay đổi
3. **Trải Nghiệm Người Dùng**: Xem xét responsive mobile và accessibility
4. **Xử Lý Lỗi**: Validate inputs và trả về thông báo lỗi có ý nghĩa
5. **Hiệu Suất**: Giữ renders và API calls hiệu quả

## Các Tác Vụ Phổ Biến

- **Thêm Tính Năng**: Xem xét cả backend (endpoint mới) và frontend (UI updates)
- **Sửa Lỗi**: Test trong môi trường phát triển và trình duyệt
- **Hiệu Suất**: Profile backend response times và frontend render times
- **Testing**: Manual testing trong trình duyệt

---

*Nhớ chạy `npm install` nếu dependencies thay đổi và `npm start` để test locally.*
